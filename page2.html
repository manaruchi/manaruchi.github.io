PROJECTS

  1. Preparation of daily cloud free snow cover dataset and its assimilation into hydrological model - The project was aimed at removing cloud pixels in Moderate Resolution Spectroradiometer (MODIS) snow cover data (MOD10C1 and MYD10C1) using a series of spatio temporal filters and Variational Interpolation (VI). The cloud free dataset was prepared for Asia over the time period of 2002-2019. The generated cloud free data was assimilated into Variable Infiltration Capacity (VIC) hydrological model to calculate the snow melt runoff in the rivers of North West Himalaya (NWH).
  2. SPIUtility: An open source tool for estimation of Standardized Precipitation Index (SPI) - Developed a open-source tool as a plugin in QGIS for handling big data problem in calculation and analysis of SPI in different timescales, an index widely used in drought monitoring. More Details...
  3. SPIDash: A web-based responsive dashboard for visualization of SPI rasters over India for the period of 1901-2018 developed using Dash, Plotly and Bootstrap. The dashboard is available here with source code explaination at my Github repo.
  4. Deep learning based cloud segmentation in AWiFS data - Developed a U-Net based CNN architecture for cloud and snow segmentation in Advanced Wide Field Sensor (AWiFS) multispectral data .
  5. Time series gridded precipitation prediction using Long Short-Term Memory Model - Developed a LSTM model based of RNN as an alternative to the computationally expensive physical models to predict precipitation in Uttarakhand state of India using ERA-5 0.25 degree precipiation data as training set.
